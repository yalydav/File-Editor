# File-Editor
## Description
The code allows you to create text documents, save them, read them and delete them.

## Example
```
1 for a new file
2 to open an existed file.
3 to delete a file
>>1
enter file name>>english
enter your text >>hello
1 for a new file
2 to open an existed file.
3 to delete a file
>>2
what's the file's name?>>hebrew
hebrew
shalom
```
